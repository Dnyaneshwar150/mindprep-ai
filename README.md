# 🧠 MindPrep

MindPrep is an AI-powered exam preparation platform that helps students break down answers into **mind maps**. Students input a question, and the AI returns structured responses in the form of nodes and child nodes, making learning more **visual**, **organized**, and **easy to revise**.

## 🚀 Features

- 📌 Ask any question — AI generates the answer.
- 🌳 Auto-structured answers using **mind maps** (main points + sub-points).
- 🖼️ Visual layout with **React Flow** for smooth node-based rendering.
- 🧭 Sidebar for:
  - Inputting your question.
  - Setting number of main points and subpoints.
- ⚡ Built with modern tech: **Next.js**, **Material UI**, and **React Flow**.
